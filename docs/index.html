Everything you need, nothing you don't

⚡

Large file support

Handles millions of rows with background loading and virtual rendering — stays responsive at any size.

⌨

Keyboard-first

Navigate and edit cells entirely with the keyboard. Arrow keys, Tab, Enter, Escape — just like a spreadsheet.

🔍

Sort & filter

Sort by any column, filter rows by value, and keep a live count of visible results.

↩

Undo / redo

Full undo/redo for all cell edits. Edit confidently and step back whenever you need.

📊

Selection stats

Select a range of cells and instantly see sum, average, min, max, and count in the status bar.

🎨

Themes

Ships with light and dark themes. Drop in any DTCG design token file for full customisation.
